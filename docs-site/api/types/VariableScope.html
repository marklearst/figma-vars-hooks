    Enum of all valid Figma variable scopes.

    Remarks

    Scopes define where a variable can be referenced or applied (e.g., fills, text content, opacity, effects). Use this to restrict variables to certain design properties in the Figma UI or API.

    Example

    const scopes: VariableScope[] = ['ALL_FILLS', 'TEXT_CONTENT']
